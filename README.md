# Robotic-Control
Pytorch implementation of several robotic control algorithm.

## Requirement
- python 3.6
- OpenCV

## Demo
* Bicycle Kinematic Model
* PID Longitude Control 
* Stanley Lateral Control
* RRT* Path Planning 
* Cubic-Spline Curve
```
$python nav_demo.py
```
